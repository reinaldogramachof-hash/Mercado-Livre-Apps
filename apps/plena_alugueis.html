Como garantir que não vou perder meus dados?

O sistema é Local-First, ou seja, seus dados ficam salvos no navegador do seu dispositivo. Para não perdê-los:

  1. Vá até Configurações no menu
  2. Clique em Baixar Backup Completo
  3. Salve o arquivo .json em local seguro
  4. Faça backup pelo menos uma vez por semana

Dica: Envie o backup para seu e-mail ou Google Drive

Como instalar como Aplicativo no celular?

O Plena Aluguéis é um PWA (Progressive Web App) e pode ser instalado direto no celular:

Android (Chrome):

Clique nos 3 pontos → "Adicionar à tela inicial"

iPhone (Safari):

Compartilhar → "Adicionar à tela de início"

Como compartilhar recibos e relatórios?

Você pode compartilhar recibos e relatórios de várias formas:

  • WhatsApp: No Dashboard, clique no ícone do WhatsApp para enviar recibo
  • PDF: Exporte o relatório completo em PDF
  • CSV: Exporte os dados para Excel/Google Sheets
  • Impressão: Imprima diretamente do sistema
