Como garantir que não vou perder meus agendamentos?

O sistema é Local-First, seus dados ficam salvos no navegador. Para não perdê-los:

  1. Vá até Configurações no menu
  2. Clique em Baixar Backup Completo
  3. Salve o arquivo .json em local seguro
  4. Faça backup pelo menos uma vez por semana

Dica: Envie o backup para seu WhatsApp ou Google Drive

Como instalar como Aplicativo no celular?

O Plena Beleza é um PWA (Progressive Web App) e pode ser instalado direto no celular:

Android (Chrome):

Clique nos 3 pontos → "Adicionar à tela inicial"

iPhone (Safari):

Compartilhar → "Adicionar à tela de início"

Como funciona o cálculo de comissões?

O sistema calcula automaticamente as comissões:

  • Cadastre o profissional com a porcentagem de comissão
  • Ao finalizar um atendimento, o sistema calcula automaticamente
  • Consulte as comissões pendentes na tela de Profissionais
  • Pague diretamente pelo sistema com um clique
